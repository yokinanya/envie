欢迎来到 Envie 用户指南！

Envie 是一个可自托管、开源的密钥管理器。它可以安全地存储 API Key、应用凭证以及其他应用敏感信息。

Envie 很容易集成到任何生产流程或开发环境中，可以作为 `.env` 文件以及 `dotenv` 这类工具的直接替代方案。

### 安装

Envie 提供 npm CLI 工具。

要在本机全局安装，请运行：

```bash
npm install -g @envie/cli
```

### 基础设置

首次使用 Envie 时，直接运行不带参数的 `envie` 命令，会进入初始化向导。

向导会帮助你完成初始配置，例如设置密钥对路径和终端自动补全。

完成初始化后，可以运行下面的命令登录：

```bash
envie login
```

现在你已经可以开始使用 Envie 了。
